Node.js


Node.js was developed by Ryan Dahl in 2009 and its latest version is v0.10.36. Node.js is an open source, cross-platform runtime environment for developing server-side and networking applications. A runtime environment for JavaScript that runs on the server.

    Node.js frameworks and tools


    2009

    • Node.js is born

    2010

    • Express one of the most simple yet powerful ways to create a web server. Its minimalist approach, unopinionated, focused on the core features of a server, is key to its success.
    • Socket.io a real-time communication engine to build network applications.

    2011

    • npm hits version 1.0 Larger companies start adopting Node.js: LinkedIn, Uber, etc.
    • hapi a very modular and simple to use configuration-centric framework for building web and services applications
