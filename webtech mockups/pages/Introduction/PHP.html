PHP


An acronym for Hypertext Preprocessor, created by Rasmus Lerdorf unleashed the first version of PHP way back in 1994. PHP is a server side scripting language that is embedded in HTML. It is used to manage dynamic content, databases, session tracking, even build entire e-commerce sites.

    PHP SUPERGLOBALS


  • $GLOBALS: It is a superglobal variable which is used to access global variables from anywhere in the PHP script.
  • $_SERVER: It stores the information about headers, paths and script locations.
  • $_REQUEST: It is a superglobal variable which is used to collect the data after submitting an HTML form. $_REQUEST is not used mostly, because $_POST and $_GET perform the same task and are widely used.
  • $_GET: Is a super global variable used to collect data from the HTML form after submitting it. $_GET super global array variable stores the values that come in the URL.
  • $_POST: It is a super global variable used to collect data from the HTML form after submitting it.
  • $_SESSION: Contains all variables stored in a user's session.
  • $_COOKIE: Contains all variables sent via HTTP cookies.
  • $_FILES: Contains all variables sent via a HTTP POST file upload.
  • $_ENV: Contains all environment variables.
